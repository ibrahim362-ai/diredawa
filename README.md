# Dire Dawa City Official Website

A comprehensive, multilingual website showcasing Dire Dawa City Administration - the Gateway of Eastern Ethiopia.

## Features

- **Multilingual Support**: Available in English and Amharic
- **Responsive Design**: Optimized for all devices
- **Modern UI/UX**: Clean, professional design with Ethiopian flag colors
- **Comprehensive Information**: City overview, administration, economy, education, and more

## Sections

- City Overview & Statistics
- Administrative Structure (Kebeles & Gandas)
- Population & Society
- Markets & Economy
- Governance & Administration
- Education Sector
- Technology & Innovation
- Sports & Recreation
- Tourism & Culture
- Contact Information

## Technologies Used

- HTML5
- CSS3 (Modern Grid & Flexbox)
- Vanilla JavaScript
- Font Awesome Icons
- Google Fonts (Inter)

## Live Demo

Visit the live website: [Dire Dawa City](https://ibrahim362-ai.github.io/diredawa/)

## Local Development

1. Clone the repository
2. Open `index.html` in your browser
3. No build process required - pure HTML/CSS/JS
